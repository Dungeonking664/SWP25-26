
GitHub Copilot (github-copilot)
================
GitHub's AI models, integrated with development tools

Models:
  - github-copilot/gpt-4o
  - github-copilot/gpt-4-turbo
  - github-copilot/claude-3-opus

Setup:
  export GITHUB_TOKEN="your-api-key-here"
  opencode run "prompt" --model github-copilot/gpt-4o
